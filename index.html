How it runs

One trigger, six feeds, one verdict

01

Schedule Trigger

Fires once daily and fans out to every ingestion node in parallel — nothing waits on anything else.

02

Six ingestion nodes, run in parallel

Each node pulls a different slice of the market: what competitors are selling, how the brand's own site is performing, and what's trending around it.

Competitor Scraper — Nykaa / Purplle / Tira GA4 Instagram Reels Meta Ads Inventory Reviews
03

Normalise & Merge

Six mismatched schemas become one dataset — same brand, same SKU, same day, no matter which platform it came from.

04

Three AI agent chains, run in parallel

Each chain specialises in one lens on the merged data, so no single agent has to reason about everything at once.

Pricing & Promotions Agent Trend & Buzz Agent Sentiment & Reviews Agent
05

Master AI Synthesis

Reconciles the three chains into a single daily read: what moved, why it matters, and what's worth acting on.
